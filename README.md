<h1 align="center">Glassmorphism Calculator </h1>

Javascript calculator with glassmorphism UI

### ✨ [Demo](https://glassmorphism-calculator.pages.dev/)

## Usage

```bash
# Install dependencies
npm install
OR
yarn

# Run dev server
npm run dev
OR
yarn dev

# Build for export and deploy
npm run build
OR
yarn build
```
